# 📚 EduLibrary

EduLibrary is a modern web application that allows users to explore and access educational books and courses online.
The platform provides an easy and organized way to browse learning resources.

## 🚀 Features

* 📖 Browse educational books and courses
* 🔍 Search functionality for books
* 🔐 User authentication (Login / Logout)
* 🌙 Dark / Light theme toggle
* 📱 Fully responsive design
* 🎨 Modern UI built with Tailwind CSS and DaisyUI

## 🛠️ Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* DaisyUI
* React Router

**Backend**

* Node.js
* Express.js
* MongoDB

**Other Tools**

* Axios
* Git & GitHub

## ⚙️ Installation

1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/EduLibrary.git
```

2️⃣ Go to the project folder

```bash
cd EduLibrary
```

3️⃣ Install dependencies

Frontend:

```bash
cd Frontend
npm install
npm run dev
```

Backend:

```bash
cd Backend
npm install
npm start
```
